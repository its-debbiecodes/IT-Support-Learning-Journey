# Day 1 -- What is IT support
IT suport involves helping users or custormers solve technical issues so they can continue their work efficiently. IT support requires good communication, problem solving and patience.

## Key skills
- Communication
- Troubleshooting
- Staying calm under pressure
- Willingness to earn

## Example scenario
IF A USER SAID THEIR COMPUTER WAS RUNNING SLOW
- Ask the user questions to understand the cause of issue.
- check task manager for background tabs.
- Check physical issue like overheating.
